# dpath

## Features

Read deep into highly nested or long xml files.

![Demo](./images/demo.gif)

Create a simple path from the nested tree structure.

![Path](./images/path.gif)

Support for namespaced and default xmlns URI's

![Namespace](./images/namespace.gif)

## Extension Settings

This extension contributes the following settings:

* `dpath.bufferSize`: Size of the buffer (KiB) to use when reading files.
* `dpath.debounce`: Debounce time in milliseconds.

**Enjoy!**
